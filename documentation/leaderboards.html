Best practice: Leaderboards

You can use the GameService module to create visual leaderboards, record a player's score, and compare the score against the player's score from previous game sessions.

Understanding leaderboards

If you haven't already done so, you might find it helpful to review theÂ leaderboards game concepts.

Updating the player's score

When the player's score changes (for example, when the player finishes the game), your game can update their score on the leaderboard by calling submitHighscore(), and passing in the leaderboard ID and the raw score value.

gameservice.submitHighscore("leaderboard_id", 1234);

Displaying leaderboards

To show the default leaderboard UI, simply call showLeaderBoard().

gameservice.showLeaderBoard("leaderboard_id")
