Best practice: Achievements

You can use the GameService module to unlock and display achievements in your game.

Understanding achievements

If you haven't already done so, you might find it helpful to review the achievements game concepts.

Unlocking achievements

To unlock an achievement, call the unlockAchievement() method and and pass in the achievement ID.

gameservice.unlockAchievement("achievement_id")

If the achievement is of the incremental type (that is, several steps are required to unlock it), call incrementAchievement() instead.

gameservice.incrementAchievement("achievment_id", 1)

You do not need to write additional code to unlock the achievement; Play Games services automatically unlocks the achievement once it reaches its required number of steps. If the achievement is hidden, it becomes automatically visible for the user once one of the methods are called.

Reveal achievements

You have the option to hide certain achievements. To show those achievements to the user, just call the revealAchievement() method.

gameservice.revealAchievement("achievement_id")

Displaying achievements

To show the player's achievements, simply call showAchievements().

gameservice.showAchievements()
